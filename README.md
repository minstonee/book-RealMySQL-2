- 주마다 한 명이 리딩하고, 해당 챕터에 대해서 공유 & 토론
    - 챕터마다 질문 뽑아서 랜덤 답변 & 토론 & 결론 정리
    - 읽으면서 인상깊었던 점, 중요하게 생각했던 것들 공유
    - 공부하면서 찾아봤던 추가적인 자료 공유
